---DAL------------------
В проекте четыре модели: Category, Product, User, Role. В Product.Url - часть пути к картинке.
Для каждой модели класс-репозиторий. 
Как единый класс доступа к бд - UnitOfWork.



---BL--------------------
Использовала AutoMapper.



---Web-------------------
Для админа после входа (email: admin@gmail.com, password:123456) товары выводятся с возможностью добавление, изменения, удаления.
При редактировании и добавлении товара можно загружать картинку. 
JS для модальных окон добавления/редактирования - CreateEditPopUp.js.
Отображение картинкок - через обработчик. Зарегистрирован в web.config в секции system.webServer. 


Использована аутентификация OWIN и ClaimsIdentity. 
Чтобы в базе хранить зашифрованный пароль, в BL использовала метод класса Crypto.HashPassword(). 
Так же можно зарегистрироваться, но только как обычный user.
JS для модальных окон логин/регистрации - ModalPopuoAuth.js.


Есть ApiController. 
На странице Index контроллера Example лежит файлик WebApi.js. На эту страницу Index можно перейти только если в url самому внести /example/index.
Отображение, редактирование, добавление(с загрузкой картинки) и удаление товара через ajax.



---Dependency------------------
Использован Autofac.
